can-eth-utils
============

Requirements:
-------------

+ gcc
+ make
+ pkg-config
+ libnl-3
+ libnl-3-dev
+ libnl-genl-3
+ libnl-genl-3-dev

Installation
------------

	make
	make install
	cd src/cegwsend/
	make
	make install

Usage
-----

look for

	man cegwctl
	man cegwsend

how to use the commands.
